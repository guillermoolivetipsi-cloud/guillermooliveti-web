2. Datos que se recopilan

Este sitio web puede recopilar los siguientes tipos de datos:

  • Datos de navegación: páginas visitadas, tiempo de permanencia, dispositivo y navegador, dirección IP (anonimizada).
  • Datos de contacto: nombre y dirección de correo electrónico, cuando el usuario inicia contacto voluntariamente.
  • Datos de agendamiento: nombre y correo electrónico proporcionados al reservar una sesión a través de Calendly.
  • Datos de suscripción: correo electrónico proporcionado al suscribirse al newsletter de NeoPsy a través de Beehiiv.

3. Finalidad del tratamiento

Los datos recopilados se utilizan exclusivamente para:

  • Prestar el servicio de psicología clínica online.
  • Gestionar el agendamiento de sesiones.
  • Enviar el newsletter de NeoPsy a quienes lo hayan solicitado expresamente.
  • Analizar el comportamiento de los usuarios en el sitio con fines estadísticos y de mejora (Google Analytics 4).
  • Mostrar publicidad relevante a usuarios potenciales (Google Ads), previa aceptación de cookies.

4. Base legal del tratamiento

El tratamiento de datos se realiza bajo las siguientes bases legales:

  • Consentimiento expreso del usuario para el uso de cookies analíticas y publicitarias.
  • Ejecución de un acuerdo de servicios, para la gestión de sesiones y comunicaciones clínicas.
  • Interés legítimo, para el análisis básico del funcionamiento del sitio.

Esta política cumple con la Ley 25.326 de Protección de Datos Personales (Argentina), la Ley 1581 de 2012 (Colombia) y la Ley 8968 de Protección de la Persona frente al Tratamiento de sus Datos Personales (Costa Rica).

Este sitio utiliza los siguientes servicios de terceros que pueden tratar datos personales bajo sus propias políticas de privacidad:

  • Google Ads (Google LLC) — publicidad online. Solo se activa con consentimiento previo del usuario. Ver política de Google.

Algunos de los servicios de terceros mencionados pueden transferir datos a servidores ubicados fuera del país de residencia del usuario. Estas transferencias se realizan con las garantías adecuadas establecidas por las normativas de protección de datos aplicables.
